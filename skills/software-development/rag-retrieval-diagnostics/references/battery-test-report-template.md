# Battery Test Report Template

Standard template for generating MD+HTML dark-theme reports from battery test results.

## Report Structure

1. Title with version and date
2. Executive summary table (PASS/WARN/FAIL/Content/Conf)
3. Per-level breakdown (Basico/Intermedio/Avanzado)
4. Detail of problems (FAILs and LOWCONFs)
5. Historical comparison table
6. Root cause analysis
7. Next steps
8. Generated files list

## HTML Styling

Dark theme matching GitHub dark mode:
- Background: #0d1117
- Text: #c9d1d9
- Headers: #58a6ff (h1), #f0883e (h2), #d2a8ff (h3)
- Tables: #161b22 with #21262d borders
- Code: #f0883e on #161b22
- Hover rows: #1c2128
